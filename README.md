### Unidad 1. Algoritmos
*¿Qué es un algoritmo?*

Es un proceso o secuecnia de pasos que se siguen para darle solucion a un problema, estos pasos deben ser sistemáticos, hiperativos y finitos.
### Lenguajes de Programación
*¿Qué es lenguaje de Programación?*

Es un conjunto de reglas y sintaxis que permite a los humanos dar instrucciones a una computadora para que realice tareas específicas.

### Concepto de Algoritmo
**1. Entradas :** Los insumos que entran para poder dar solucion al problema (números, nombres, fechas, imágenes, archivos de audio, señales de onda, entre otros)

**2. Procesos :** Las acciones que se encargan de procesar los datos de entrada con el propósito de devolver los resultados esperados.

**3. Salidas :** son las diferentes formas en que se presentan los resultados del procesamiento de los datos que entraron.

<img width="1000" height="388" alt="image" src="https://github.com/user-attachments/assets/ff45dbf5-8d6f-4e4c-a434-590dfb49e5df" />

### Condiciones del algoritmo
*1. debe resolver un problema*
*2. Debe ser finito*
*3. interactuar con datos externos y/o variables*
### Variables y constantes
- **Variable**,  Son los espacios en memoria que almacenan los datos que serán procesados para obtener los resultados esperados, *(talla, direccion de una casa, edad.)*
- **Constante**, Valores que resultan invariables *(Un numero, una formula, el numero PI)*
<img width="305" height="165" alt="image" src="https://github.com/user-attachments/assets/53a29b90-6b4b-40c3-aed3-5dfbba1368ea" />

### Operadores
1. *Aritmeticos*, operaciones matemáticas básicas.
2. *Relacionales*,  expresiones condicionales a partir de la comparación de dos variables y/o valores
3. *Lógicos*, se usan para evaluar expresiones condicionales a partir de la unión de varios valores y/o variables

## Ejemplo de codigo  python
```python
Programa que calcula en qué año cumplirás 100 años
nombre = input("¿Cuál es tu nombre? ")
edad = int(input("¿Cuántos años tienes? "))
Calculamos el año en que cumplirá 100 años
anio_actual = 2025
anios_para_100 = 100 - edad
anio_cumple_100 = anio_actual + anios_para_100
Mostramos el resultado
print(f"Hola {nombre}, cumplirás 100 años en el año {anio_cumple_100}.")
```
